# Node.js / TypeScript Project Template

A `CLAUDE.md` starter for Node.js projects using TypeScript, Vitest, ESLint, and Prettier. Tells Claude your toolchain, conventions, and guardrails so it can be effective from the first message.

## What's included

- Tech stack declaration (Node 22+, TypeScript 5+, Vitest, ESLint, Prettier)
- Standard project structure
- Dev commands Claude can invoke directly
- TypeScript strictness expectations
- Testing guidelines (unit vs integration boundary)
- Explicit constraints (no CommonJS, no direct-to-main commits)

## Installation

```bash
/plugin install nodejs-project@agentic-plugins-marketplace
```

Edit `CLAUDE.md` after installation to fill in your project's specific details.
