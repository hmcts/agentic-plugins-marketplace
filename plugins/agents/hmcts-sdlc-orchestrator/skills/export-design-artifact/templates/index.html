Pick by intent

If you're producing…UseBecause
A code reviewAudit ConsoleSeverity + diff is the universal review vocabulary
An explainer / teaching docAuroraCalm, spacious, visual — built to be read slowly
A delivery planRoadmapPhases + progress answer "where are we"
A debug / RCA write-upDossierEvidence → hypothesis → verdict mirrors investigation
A user / process flowJourneySwimlanes + emotion show who-does-what and how it feels
A cutover / migrationRunbookGates + rollback + risk = safe, reversible execution
A design decision (3+ options)Decision RecordMatrix + weighted score makes the call defensible
A two-way trade-offVersusMirrored bars make the winner obvious at a glance
A system/architecture mapBlueprintSchematic language reads as authoritative structure
A status / rollout dashboardMission ControlKPIs + rings + health convey state instantly
An event / API sequenceSignal TraceLifelines + flow show order and async hops
